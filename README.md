# Tripleten web_project_homeland

Descripción
En este proyecto, "De Patria a Patria", contruimos la página de inicio de la exhibición, una exhibición en la galería de arte Triple Ten. La página web de la exhibición debe ser responsiva, lo que significa que el diseño tendrá que adaptarse, hacer que tu sitio se vea bien en todos los dispositivos. Se trata de un sitio que muestra las ciudades natales de algunos empleados de TripleTen: "Un viaje épico de Kentucky a Burundi pasando por Gales y Ucrania".

La funcionalidad
-Trabajar con diseños en un editor gráfico llamado Figma.
-Utilizar consultas de medios (media queries) para establecer parámetros para diferentes tamaños de pantallas.
Diseño responsivo que se adapta a tres resoluciones principales:
Desktop: 1280px
Tablet: 768px
Mobile: 320px
-Trabajar con Git y la línea de comandos de manera más eficiente.

Tecnologias
-html
-css:
-flexbox
-metodologia BEM
-media queries
-agregar fuentes @font face
